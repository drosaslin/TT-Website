個人課程
Course Description

依照學員的年齡和技能程度，教練先施實動作技術診斷及規畫課程目標，訓練過程給予即時的動作回饋與事後動作分析技術找出問題並給予建議。將動作(世界級好手或台灣好手)與學員的前、後動作分析做比較，強化正確動作觀念，避免錯誤動作造成運動傷害，學習者能快速修正動作 ，更有效率的提升運動表現。強化各項技術、步伐及技戰術的結合訓練，模擬比賽情境。課程資料建檔上傳，學員可以隨時隨地的複習動作，可作為下次修改動作的依據，提升學習效果。

Pricing Information

甲組教練 ：一對一教學 單次課程 $1080/1hr (1堂)。一期課程 $10200/10hr (10堂)。

青少年國手教練：一對一教學 單次課程 $1200/1hr (1堂)。一期課程 $11300/10hr (10堂)。

成年國手教練：一對一教學 單次課程 $1300/1hr (1堂)。一期課程 $12000/10hr (10堂)。

Helpful Information

1. 提升桌球技術、步伐及技戰術組合運用

2. 對球的注意力判斷與預期能力

3. 區辨對手優點和缺點

4. 正確的動作觀念、動作分析及動作修正

5. 動作影像即時回饋

Don't hesitate to contact us if you have more questions

團體A課程
Course Description

教練依照學員的技術診斷或台灣桌球積分賽、桌球愛好者聯盟作為參考依據，了解學員的需求，並規畫適當的課程。課程內容包含各單項技術的強化及戰術的結合訓練，透過即時動作影像回饋，再配合動作分析，更有效率的修正動作及後續建檔追蹤有助於學習效果，享受學習的樂趣，提升運動表現。

Coach Level

青少年國手(曾經參加獲得全國比賽名次並當選青少年國手)

Group Size

2位即可開班，最多3位

Course Time

首次課程~4個月內

Recommended Experience

台灣桌球積分賽(901以上)、桌球愛好者聯盟(2001以上)。

基礎技術穩定，主要強化技戰術運用與整合及模擬比賽或特定動作技術練習。

Pricing Information

教練一對二教學 單次課程 $ 780元/1hr(1堂)。一期課程 $ 7000/10hr (10堂)。

教練一對三教學 單次課程 $ 840元/1.5hr (1堂)。一期課程 $7500/15hr(10堂)。

Helpful Information

1. 提升桌球技術、步伐及技戰術組合運用

2. 區辨對手優點和缺點

3. 對球的注意力判斷與預期能力

4. 正確的動作觀念、動作分析及動作修正

5. 動作影像即時回饋

Remarks

選擇課程若學員若無任何積分賽分數，則由教練實施技術評估及安排規劃。

學員可以選擇教練分級或依照教練實施技術評估及安排規劃。

若課程只有1人，則改成1對1個人教學，課程時間為1hr，費用比照1對1。

報名時若只有2位學員可選擇雙打或單打。

Don't hesitate to contact us if you have more questions

學齡前團體課程
Course Description

依照學員的技術程度，教練給予循序漸進的課程規劃，以趣味化的遊戲課程為主，各項基礎技術動作強化及組合訓練，搭配基本運動能力訓練來強化學員的身體及心智的發展，鼓勵彼此人際互動、表達能力，促使孩童全面性的發展。

Group Size

平均上課帶2-4位學員

Pricing Information

單次課程 $ 700元/1.5hr(1堂)。一期課程 $ 6500/15hr (12堂) 。

若課程只有1人，則改成1對1個別教學，一期課程 $ 8000/10hr (10堂)。

Helpful Information

1. 桌球基礎技術、動作、步伐

2. 基本運動能力(敏捷、協調、平衡、速度、反應)

3. 認知 (注意力、工作記憶、抑制控制)、統合感覺

4. 培養運動習慣、勇於展現肢體，並樂於參與人際互動、口語表達

5. 引導模仿及協調控制動作技能

6. 適當增加身體活動量

Don't hesitate to contact us if you have more questions
